About Me

I’m a software engineer with 20+ years of experience across the full software development lifecycle. Over 15 years at one organization, I grew from Software Engineer to Senior Software Engineer to Software Development Manager, leading architecture and delivery for cloud, mobile, and IoT systems — designing RESTful APIs and microservices in Java and Spring, migrating legacy applications to React and Node.js, and driving a cloud migration from AWS to Azure as technical lead and Scrum Master.

Since 2021, I’ve worked as a freelance full-stack engineer and project manager, and I currently volunteer remotely as a software developer, building Python and Qt tools for a global production team. That work is also where I started learning AI development hands-on — building AI-integrated pipelines, applying prompt engineering, and using AI-assisted workflows to modernize legacy code — and I’ve continued building on those skills since. I recently completed a full-stack bootcamp to sharpen my skills in React, Node.js, and modern JavaScript, and I’m now looking for my next full-time role where I can bring two decades of engineering and technical leadership experience to a team.
